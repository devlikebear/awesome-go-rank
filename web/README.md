## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Data Source

The application reads repository data from `../public/data/repos.json`, which is generated by the Go backend (`awesome-go-rank`).

To generate the latest data:

```bash
# From project root
cd ..
GITHUB_TOKEN=your_token go run cmd/main.go
```

This will create/update `public/data/repos.json` with the latest repository information.

## Deployment

### Vercel (Recommended)

1. Import your GitHub repository to Vercel
2. Set the root directory to `web/`
3. Framework preset: Next.js
4. Build command: `npm run build`
5. Output directory: `out`

### Manual Deployment

```bash
npm run build
# Deploy the 'out' directory to your hosting provider
```
