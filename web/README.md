# Awesome Go Ranking - Web Application

Modern, responsive web interface for browsing and discovering Go repositories from [awesome-go](https://github.com/avelino/awesome-go).

## Features

- 🔍 **Real-time Search** - Instantly search through thousands of repositories
- 🏷️ **Category Filters** - Browse by specific categories
- ⭐ **Smart Sorting** - Sort by stars, forks, or recent updates
- 🎨 **Dark Mode** - Beautiful dark/light theme support
- 📱 **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Fast Performance** - Static site generation for lightning-fast loading

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Lucide Icons** - Beautiful icon library

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Data Source

The application reads repository data from `public/data/repos.json` inside this `web/` directory. This is the single canonical JSON path generated by the Go backend (`awesome-go-rank`).

To generate the latest data:

```bash
# From project root
cd ..
GITHUB_TOKEN=your_token go run cmd/main.go
```

This will create/update `web/public/data/repos.json` with the latest repository information.

## Deployment

### Vercel (Recommended)

1. Import your GitHub repository to Vercel
2. Set the root directory to `web/`
3. Framework preset: Next.js
4. Build command: `npm run build`
5. Output directory: `out`

### Manual Deployment

```bash
npm run build
# Deploy the 'out' directory to your hosting provider
```
